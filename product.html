Widget - Produto

Esse Widget tem como função principal aprensentar um produto, com foto, titulo e breve descritivo, alem de formatar um link para a página do mesmo.

É possivel setar via parametro a quantidade de imagens que serão exibidas do produto, variando entre 1 ou 2.

Apresentação

Versão Preview
1 imagem
2 imagens

Markup Base

Esse widget precisa de atributos em sua tag obrigatoriamente.

  
  <landing-product
    title="camiestas"
    desc="Lorem ipsum dolor sit amet, consectetur adipisicing elit. Laboriosam aperiam cum delectus deleniti."
    image="https://basico-shopify.imgix.net/s/files/1/1324/3705/products/20170321-M_camiseta-gola_v-preto-p-1.jpg?format=compress&w=800"
    link="/products/jaqueta-tyvek-masculina-branco-preto"
    label="comprar camiseta">
  </landing-product>

Aplicações

Esse Widget só tem uma aplicação e o layout é o mesmo para todos os breakpoints.

Seus parametros são:

Parametro Obrigatório Descrição
title não titulo do widget, que deve, provavelmente remeter ao produto
desc não breve descritivo do widget (produto)
image sim imagem principal do produto
image2 não imagem secundaria não obrigatoria
link sim link para a página do produto
label sim label do botão do widget
