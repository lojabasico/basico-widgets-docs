Widget - Post

Esse Widget tem como função principal apresentar uma outra landing page, de forma sucinta e que possibilite uma apresentação listada de outras páginas que façam parte do mesmo grupo de landing pages, ex.: Páginas de "Sou Básico"

Apresentação

Versão Preview
Desktop
Mobile

Markup Base

Esse widget precisa de atributos definidos em sua tag, para manter a estrutura HTML inalteravel.

Esse widget não necessita de nenhum conteúdo interno de sua tag de aplicação!


  <landing-post image="history-1.jpg"
    title="luna buschianelli"
    desc="Lorem ipsum dolor sit amet, consectetur adipisicing elit."
    link="https://basicoteste.com/pages/historias">
  </landing-post>

Attributos

Código Nome Descrição Exemplo
image Foto Imagem que irá mostrar no widget https://basico-shopify.imgix.net/s/files/1/2316/0905/files/history-1.jpg
title Titulo Titulo do widget luna buschianelli
desc Descrição Breve texto corrido que aparece após o título. Lorem ipsum dolor sit amet, consectetur adipisicing elit. Rem possimus dicta fugiat modi libero illum beatae.
link Link Url para qual o widget aponta. https://basicoteste.com/pages/historias

Aplicações

Esse Widget só tem uma aplicação, com uma configuração de responsividade ja formatada.
