Widget - Listener

Esse Widget possibilita mostrar conteúdo dinâmico com base no comportamento de outro Widget

Na imagem acima, as 3 opções representam componentes do tipo "Radio", que emitem um evento especial LandingRadio/grupo, toda vez que o usuário clica em uma opção.

O listener representa o texto que está logo abaixo. Você pode perceber que conforme o usuário escolhe uma opção, o texto "muda".
Isso é o que chamamos de "reagir" a uma interação de outro componente.

O componente Listener "reage" a uma mudança de um outro componente (no caso o Radio), e pode mostrar informações ou ativar comportamentos dependendo da interação do usuário.

Esse Widget só é compatível com componentes que emitem eventos.

Markup Base

  
  <landing-listener eventname="algum_evento">
    <template scope="props">
      ...
    </template>
  </landing-listener>

Attributos

Esse widget possui apenas 1 atributo.

Código Nome Descrição Exemplo
eventname Nome do Evento É o nome do evento que o componente que você quer reagir emite (para saber qual valor usar, você deve consultar a documentação do componente compatível que você quer interagir) LandingRadio/grupo

Aplicações

Esse Widget tem infinitas aplicações, então para facilitar vou dar um exemplo prático seguindo como base o exemplo do Widget dos 3 Preços.

  
    <!-- COMPONENTE LANDING-RADIO -->
    <landing-radio name="teste" label="R$ 98" value="98"></landing-radio>
    <landing-radio name="teste" label="R$ 118" value="118"></landing-radio>
    <landing-radio name="teste" label="R$ 193" value="193"></landing-radio>

    <!-- COMPONENTE LANDING-LISTENER -->
    <!-- O componente listener está interagindo com o componente LandingRadio. -->
    <!-- Aqui, para cada opção do LandingRadio, existe um texto específico que é exibido ao usuário -->
    <landing-listener eventname="LandingRadio/teste">
      <template scope="props">
        <template v-if="props.eventdata.value == '98'">
          <h2>onde estamos</h2>
          <p>por trabalharmos com importação - entre outras coisas - estamos sujeitos a flutuações que impactam nossos custos, do câmbio e materiais à logística. este valor atualmente cobre apenas os custos de produção e operação</p>
        </template>
        <template v-if="props.eventdata.value == '118'">
          <h2>onde vamos</h2>
          <p>precisamos rever nossos preços para mantermos uma equação saudável, mas ainda com o compromisso de oferecer básicos de qualidade a um valor justo. este preço cobre custos básicos de operação e comunicação, com margem para seguirmos investindo em novos produtos.</p>
        </template>
        <template v-if="props.eventdata.value == '193'">
          <h2>onde está o mercado</h2>
          <p>este é o preço médio praticado no mercado para produtos de qualidade equivalente e composição 100% algodão pima. otimizamos nossa operação e eliminamos intermediários para que você não pague este valor por aqui.</p>
        </template>
      </template>
    </landing-listener>
