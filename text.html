Widget - Text

Esse Widget tem como função principal inserir, já formatados todos os tipos de texto nas páginas, como titulos, subtitulos e paragrafos.

O widget formata os seguintes markups:

  • h1
  • h2
  • h3
  • h4
  • h5
  • h6
  • p

Markup Base

Esse widget não precisa de atributos em sua tag.

  
  <landing-text>
    <h1>Tela em Branco</h1>
    <p>Lorem ipsum dolor sit amet, consectetur adipisicing elit.</p>
  </landing-text>

Aplicações

Esse Widget só tem uma aplicação, com uma configuração de responsividade ja formatada.

É importante notar, que ele obece uma estrutura HTML padrão para formatação de seus componentes (h1, h2, p, etc...).

  
  <landing-text>
    <h1>Tela em Branco</h1>
    <p>Lorem ipsum dolor sit amet, consectetur adipisicing elit.</p>
  </landing-text>

Existem classes que sobrescreverm as classes default, para poder alterar os tamanhos em breakpoints diferentes. Essas classes seguem um padrão onde deve ser colocado o numero do breakpoint na classe.

Ex.: text-620-ultra, text-920-ultra
