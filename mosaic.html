Widget - Mosaico

Esse Widget tem como função principal arrumar imagens em formato de mosaico. Ele é formado automaticamente, passando a lista de imagens, mas funciona somente com tags img

Esse widget só funciona com tags <img>

Apresentação

Versão Preview
Desktop
Mobile

Esse widget não precisa de atributos em sua tag.

Aplicações

Esse Widget só tem uma aplicação, com uma configuração de responsividade ja formatada.

No breakpoint de DESKTOP o mosaico é formatado com 3 colunas e em MOBILE com 2.
