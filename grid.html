Widget - Grid

Esse Widget tem como função principal limitar componentes ao grid do layout.

Ele não é visivel no front da página, porem é um widget essencial para a formatação da página.

Outro detalhe importante desse widget é que ele é recursivo, ou seja, é possivel inserir uma grid dentro de outra, possibilitando assim o alinhamento dos componentes dentro de suas respectivas limitações de layout.

Detalhes do grid:

  • total de colunas: 36
  • gutter: 14px
  • margens laterais: 14px

Markup Base

Esse widget precisa de atributos definidos em sua tag, para manter a estrutura HTML inalteravel.

  
  <grid columns="36">
    <grid columns="36" desk_columns="8">
      ...
    </grid>
  </grid>

Attributos

Código Nome Descrição Exemplo
columns Colunas (default) Imagem que irá mostrar no widget 36
mobile_columns Colunas para breakpoint de MOBILE Numero de colunas que serão aplicadas somente nos breakpoints de MOBILE 36
tablet_columns Colunas para breakpoint de TABLET Numero de colunas que serão aplicadas somente nos breakpoints de MOBILE 18
desk_columns Colunas para definições de TABLET e DESKTOP Numero de colunas que serão aplicadas somente nos breakpoints de DESKTOP e TABLET 9

Aplicações

Esse Widget só tem uma aplicação.

O conteúdo interno desse widget serão os outros widgets ou HTML, lembrando que o intuito deste é limitar esse conteúdo interno dentro do grid do tema.

  
  <grid columns="36">
    <grid columns="36" desk_columns="8">
      <landing-text>
        <h1>Tela em Branco</h1>
        <p>Lorem ipsum dolor sit amet, consectetur adipisicing elit.</p>
      </landing-text>
    </grid>
  </grid>
